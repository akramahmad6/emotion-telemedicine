# EmoCare - Emotion-Aware Telemedicine System

An AI-powered telemedicine system that detects patient emotions from text messages.

## Features
- Real-time emotion detection (Positive/Negative/Stress/Neutral)
- Doctor dashboard with live alerts
- Analytics dashboard with charts
- Firebase real-time database

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Database: Firebase Realtime Database
- AI: NLP Sentiment Analysis

## How to Run
1. Open frontend/index.html with Live Server
2. Run: cd backend && python app.py
